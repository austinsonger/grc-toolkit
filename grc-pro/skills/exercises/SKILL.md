# Exercises

Generate and facilitate tabletop exercise scenarios for incident response (IR) and contingency planning (CP) teams.

## Commands

| Command | Description |
|---------|-------------|
| `/grc:tabletop-scenario` | Generate IR/CP tabletop exercise templates with injects and discussion questions |

## When to Use

Use this skill when the user needs to:
- Plan a tabletop exercise for incident response or business continuity
- Generate realistic scenario injects and discussion questions
- Prepare for annual FedRAMP contingency plan or IR plan testing requirements
