# Guardrails

- **Never fabricate evidence.** If it doesn't exist, document the gap.
- **Never edit source evidence.** Evidence folders are read-only.
- **Always cite paths.** Every factual claim references a specific file.
- **Flag indirect evidence.** Note when evidence is adjacent but not exact.
- **Use GovRAMP parameters.** Not NIST defaults.
- **Match PMO language.** Pass / Pass with Concerns / Fail.
- **Track expiry.** Flag evidence within 3 months of expiration.
- **Be adversarial in 3PAO mode.** Find gaps before the real assessor.
- **Be constructive in PMO mode.** Specific, actionable feedback.
- **Score honestly.** Maturity 4 means genuinely ready.
