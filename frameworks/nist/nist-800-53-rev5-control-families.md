## Key Controls Detail

### AC – Access Control

| Control | Name | Description |
|---|---|---|
| AC-1 | Policy and Procedures | Establish access control policy and procedures |
| AC-2 | Account Management | Manage system accounts including establishing, activating, modifying, reviewing, disabling, and removing accounts |
| AC-3 | Access Enforcement | Enforce approved authorizations for logical access |
| AC-4 | Information Flow Enforcement | Enforce approved authorizations for controlling information flows |
| AC-5 | Separation of Duties | Separate duties of individuals to reduce risk of malevolent activity |
| AC-6 | Least Privilege | Employ least privilege principle, allowing only authorized accesses necessary to accomplish assigned tasks |
| AC-7 | Unsuccessful Logon Attempts | Enforce a limit on consecutive invalid logon attempts |
| AC-11 | Device Lock | Prevent further access after defined period of inactivity |
| AC-12 | Session Termination | Automatically terminate sessions after defined conditions |
| AC-14 | Permitted Actions Without Identification | Identify actions that can be performed without identification or authentication |
| AC-17 | Remote Access | Establish usage restrictions and implementation guidance for remote access |
| AC-18 | Wireless Access | Establish usage restrictions for wireless access |
| AC-19 | Access Control for Mobile Devices | Establish usage restrictions for mobile devices |
| AC-20 | Use of External Systems | Establish terms for authorized use of external information systems |

### AU – Audit and Accountability

| Control | Name | Description |
|---|---|---|
| AU-2 | Event Logging | Identify the types of events that the system is capable of logging |
| AU-3 | Content of Audit Records | Ensure audit records contain sufficient information |
| AU-6 | Audit Record Review, Analysis, Reporting | Review and analyze system audit records |
| AU-9 | Protection of Audit Information | Protect audit information and tools from unauthorized access |
| AU-11 | Audit Record Retention | Retain audit records for 3 years minimum (federal) |
| AU-12 | Audit Record Generation | Provide the capability to generate audit records |

### IA – Identification and Authentication

| Control | Name | Description |
|---|---|---|
| IA-2 | Identification and Authentication (Organizational Users) | Uniquely identify and authenticate organizational users |
| IA-2(1) | Multi-Factor Authentication for Privileged Accounts | Implement MFA for privileged accounts |
| IA-2(2) | Multi-Factor Authentication for Non-Privileged Accounts | Implement MFA for non-privileged accounts |
| IA-4 | Identifier Management | Manage information system identifiers |
| IA-5 | Authenticator Management | Manage system authenticators |
| IA-8 | Identification and Authentication (Non-Organizational Users) | Identify and authenticate non-organizational users |
